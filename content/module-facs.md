# 流式细胞术（FACS）

> **重要性**：⭐⭐⭐ 免疫学研究的"眼睛"，用于分析和分选细胞亚群

---

## 一、它是什么？（定义）

### 1.1 基本定义

**流式细胞术**（Flow Cytometry）是一种**在液流中检测细胞物理和化学特性**的技术。它就像给细胞做"安检"——让细胞一个个流过检测器，同时测量多个参数。

**FACS** = Fluorescence-Activated Cell Sorting（荧光激活细胞分选），不仅能检测，还能把想要的细胞"抓"出来。

### 1.2 核心原理

```
细胞悬液 → 逐个流过激光束 → 散射光+荧光检测 → 数据分析
                    ↓
               （也可以分选到不同管子）
```

---

## 二、检测的参数

### 2.1 散射光信号（无需染色）

| 参数 | 测量内容 | 应用 |
|------|----------|------|
| **FSC** (Forward Scatter) | 细胞大小 | 大小差异 |
| **SSC** (Side Scatter) | 细胞颗粒度 | 复杂程度 |

### 2.2 荧光信号（需要抗体染色）

| 荧光通道 | 常用荧光素 | 颜色 |
|----------|------------|------|
| FITC | FITC | 绿色 |
| PE | PE, PE-Cy7 | 橙红色 |
| APC | APC, APC-Cy7 | 红外色 |
| PerCP | PerCP-Cy5.5 | 深红色 |

## 三、在潘组研究中的应用

### 3.1 Bregs的鉴定

在Yuan 2019论文中，Bregs通过以下表型鉴定：

| 标志物 | 表型 | 意义 |
|--------|------|------|
| CD19 | CD19+ | B细胞标识 |
| CD5 | CD5+ | B1细胞/Bregs特征 |
| CD1d | CD1dhi | 高表达，Bregs特征 |
| IL-10 | IL-10+ | 功能性标记 |

**FACS分析策略：**
```
第一步：FSC/SSC圈出淋巴细胞群
         ↓
第二步：CD19+圈出B细胞
         ↓
第三步：CD5+CD1dhi圈出Bregs
         ↓
第四步：IL-10+显示功能性Bregs
```

### 3.2 分选实验

当需要获得纯化的B细胞进行体外实验时，使用FACS分选：

```
分选目标：CD19+CD5+CD1dhi Bregs
分选模式：Purity（高纯度模式）
纯度要求：>95%
```

## 四、常用对照设置

### 4.1 必需对照

| 对照 | 目的 |
|------|------|
| **未染色对照** | 设定阴性基线 |
| **FMO对照** | 设定圈门边界 |
| **同型对照** | 评估非特异性结合 |
| **单染对照** | 荧光补偿 |

### 4.2 补偿原理

当多个荧光素发射光谱重叠时，需要"补偿"：

```
PE溢出到FITC通道 → 从FITC信号中减去PE的干扰
FITC溢出到PE通道 → 从PE信号中减去FITC的干扰
```

## 五、实验步骤

### 5.1 样本制备

1. **组织消化**：酶消化或机械研磨
2. **细胞计数**：调整浓度至1×10^7 cells/mL
3. **抗体染色**：4°C避光30min
4. **洗涤**：PBS洗2次
5. **重悬**：流式缓冲液重悬

### 5.2 圈门策略

```
FSC-A vs SSC-A → 淋巴细胞门
         ↓
FSC-A vs FSC-H → 单细胞门（去除双细胞）
         ↓
CD19+ → B细胞门
         ↓
CD5 vs CD1d → Bregs门
```

## 六、技术的优缺点

### 6.1 优点

| 优点 | 说明 |
|------|------|
| **多参数** | 可同时检测10+个参数 |
| **高通量** | 每秒可检测数千个细胞 |
| **定量** | 获得细胞群体比例数据 |
| **分选功能** | 可获得纯化细胞 |

### 6.2 局限性

| 局限 | 解决方案 |
|------|----------|
| 样本需要单细胞悬液 | 充分消化组织 |
| 死细胞干扰大 | 加Live/Dead染色 |
| 抗体成本高 | 优化抗体用量 |

## 七、面试重点

### Q1: 流式细胞术的原理是什么？

> 答案要点：
> - 细胞逐个流过激光束
> - 检测散射光（大小、颗粒度）和荧光
> - 可同时检测多个参数

### Q2: 如何鉴定Bregs？

> 答案要点：
> - 表型：CD19+CD5+CD1dhi
> - 功能：产生IL-10
> - 通过流式多色染色鉴定

### Q3: 什么是FSC和SSC？

> 答案要点：
> - FSC（前向散射光）：反映细胞大小
> - SSC（侧向散射光）：反映细胞颗粒度
> - 可用于初步细胞分群
