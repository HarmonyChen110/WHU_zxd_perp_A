## 一、ELISA（酶联免疫吸附试验）

### 1.1 基本原理

**ELISA** (Enzyme-Linked Immunosorbent Assay) 是一种**用抗体检测抗原**的高灵敏度方法。它就像用"磁铁"（抗体）去捕捉特定的"金属"（抗原）。

### 1.2 常用类型

| 类型 | 原理 | 应用 |
|------|------|------|
| **直接ELISA** | 抗原-抗体-酶标二抗 | 简单快速 |
| **夹心ELISA** | 捕获抗体-抗原-检测抗体 | 特异性高 |
| **竞争ELISA** | 竞争结合 | 小分子检测 |

### 1.3 在潘组中的应用

**检测细胞因子水平：**
```
B细胞 + ManLAM → 上清液 → ELISA检测IL-10
                          ↓
                     定量测定(pg/mL)
```

**操作要点：**
- 标准曲线：至少5个浓度点
- 样本稀释：确保在线性范围内
- 复孔检测：减少误差

## 二、Western Blot（免疫印迹）

### 2.1 基本原理

**Western Blot** 用于检测**蛋白质表达和修饰**。就像给蛋白质拍"身份证照"。

### 2.2 实验流程

```
样品制备 → SDS-PAGE电泳 → 转膜 → 封闭 → 一抗 → 二抗 → 显影
```

### 2.3 在潘组中的应用

**检测STAT3磷酸化：**
```
B细胞 + ManLAM → 提取蛋白 → Western Blot
                              ↓
                       p-STAT3 (Tyr705)
                       total STAT3
                       β-actin (内参)
```

**关键抗体：**
- p-STAT3 (Tyr705): 检测STAT3激活
- STAT3: 检测总STAT3
- IL-10: 检测细胞因子表达
- β-actin/GAPDH: 内参对照

---

## 三、qRT-PCR（实时定量PCR）

### 3.1 基本原理

**qRT-PCR** 用于检测**mRNA表达水平**。就像用"放大镜"去看基因表达。

### 3.2 实验流程

```
RNA提取 → 反转录 → qPCR扩增 → 数据分析
```

### 3.3 在潘组中的应用

**检测IL-10 mRNA表达：**
```
B细胞 + ManLAM → 提取RNA → qRT-PCR
                            ↓
                     IL-10 mRNA水平
                     (相对GAPDH)
```

**常用分析方法：**
- ΔCt法：样本vs内参
- ΔΔCt法：处理组vs对照组

---

## 四、免疫共沉淀（Co-IP）

### 4.1 基本原理

**Co-IP** 用于研究**蛋白质相互作用**。就像用"诱饵"蛋白去"钓"它的"伙伴"。

### 4.2 实验流程

```
细胞裂解 → 抗体孵育 → 磁珠捕获 → 洗涤 → Western检测
```

### 4.3 应用场景

**研究信号复合物：**
- STAT3与其他转录因子的相互作用
- 受体-接头蛋白复合物

---

## 五、免疫荧光

### 5.1 基本原理

用**荧光标记的抗体**在组织/细胞中检测抗原定位。

### 5.2 在潘组中的潜在应用

- B细胞在组织中的分布
- STAT3核转位检测
- 与免疫组化结合的定位研究

## 六、细胞培养

### 6.1 B细胞培养条件

| 参数 | 要求 |
|------|------|
| 培养基 | RPMI-1640 + 10% FBS |
| 温度 | 37°C，5% CO₂ |
| 密度 | 1×10⁶ cells/mL |
| 刺激物 | ManLAM, LPS, anti-IgM |

### 6.2 体外实验设计

```
Day 0: 分离B细胞（磁珠分选）
         ↓
Day 1: 加入刺激物（ManLAM）
         ↓
Day 2-3: 收集上清/细胞
         ↓
检测: ELISA (细胞因子)
     qRT-PCR (mRNA)
     Western (蛋白)
     FACS (细胞亚群)
```

## 八、实验设计注意事项

### 8.1 对照设置

| 对照类型 | 目的 |
|----------|------|
| 空白对照 | 基线水平 |
| 阴性对照 | 非特异性效应 |
| 阳性对照 | 系统验证 |
| 技术重复 | 精确度 |
| 生物学重复 | 普遍性 |

### 8.2 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 高背景 | 抗体浓度过高 | 稀释抗体 |
| 无信号 | 抗体失效 | 更换抗体 |
| 重复性差 | 操作不一致 | 标准化流程 |

## 九、面试重点

### Q1: 如何检测IL-10的产生？

> 答案要点：
> - ELISA检测上清中的分泌蛋白
> - qRT-PCR检测mRNA水平
> - FACS检测胞内细胞因子

### Q2: Western Blot的原理是什么？

> 答案要点：
> - 蛋白分离（电泳）→ 转膜 → 抗体检测
> - 可检测蛋白表达和修饰
> - 需要合适的内参

### Q3: 如何设计一个好的体外实验？

> 答案要点：
> - 设置合适的对照
> - 重复实验确保可靠性
> - 选择合适的检测方法
> - 注意细胞状态和培养条件
